# Tevin Portfolio

A personal portfolio website showcasing my projects, skills, and experience as a developer.

## About

This portfolio website serves as a central hub to explore my work and learn more about my journey in web development. Built with clean, modern design principles, it provides an engaging way for visitors to connect with my professional profile.

## Features

- **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- **Project Showcase** - Highlights of my best work and technical projects
- **Interactive UI** - Smooth navigation and engaging user experience
- **Clean Codebase** - Well-structured HTML, CSS, and JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Getting Started

To view this portfolio locally:

1. Clone the repository
   bash
   git clone https://github.com/ChuloGlitch/tevin-portfolio.git
   

2. Navigate to the project directory
   bash
   cd tevin-portfolio
   

3. Open `index.html` in your web browser
